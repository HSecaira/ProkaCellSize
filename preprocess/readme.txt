Directory containing the notebooks and data necessary for preprocessing and taxonomic assignment. 

Files and directories:

*process*.ipynb	Notebooks containing the pre-processing for cell size data
*taxonomy*.ipynb	Notebooks containing the taxonomic assignment for cell size data
BMvol1.csv, BMvol4.csv, BMvol5.csv, Small_and_giant_microbes.xlsx, New_Data.csv, cyanobacteria_data.csv, data_manually.tsv	Files containing raw data of cell size collected from Bergey's manual, BacDIve, PCC, and research articles
data_BMvol1.tsv, data_BMvol4.tsv, data_BMvol5.tsv, data_cyanobacteria.tsv, data_cyanobacteria_taxID.tsv, data.tsv File pre-processed
*.xz	Files from NCBI taxonomy database (downloaded November 2021)
annot/	Output directory for taxonomic annotation
