Directory containing the notebooks, scripts, and data necessary to calculate the codon bias.

Files and directories:
cbias_corrected.tsv	TSV file containing the codon usage bias (\deltaENC' and MILC) calculated for all the genomes in the Web of Life project
codon_bias.ipynb	Notebook containing R code to calculate the usage codon bias for all the genomes in the Web of Life project
codBias/	Directory contains all the scripts and data necessary to identify the set of Highly Expressed Genes and concatenated CoDing Sequences and HEGs into a single fasta file.
