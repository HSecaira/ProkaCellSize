Directory containing the notebooks necessary to apply evolutionary models to cell size data.

Files and directories:
models_evolution.ipynb	Notebook with R code to fit and select the best evolutionary model to cell size data.
models_evolution_size_metrics.ipynb	Notebook with R code to fit and select the best evolutionary models to cell size data using multiple metric for size.
rescale_tree.ipynb	Notebook to rescale the phylogenetic tree used for evolutionary models.
