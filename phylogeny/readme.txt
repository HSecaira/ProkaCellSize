Directory containing the data and notebooks of phylogenetic "placement" of cell size data into the Web of Life reference tree.

Files and directories:

phylogeny.ipynb	Notebook for phylogenetic placement of BacDive data
phylogeny_cyanobacteria.ipynb	Notebook for phylogenetic placement of PCC data
phylogeny_full.ipynb	Notebook for phylogenetic placement of all data (BacDive, Bergey's manual, articles, and PCC)
genome.tsv	genome metadata of each species in the phylogeny of the WoL project
tree.nwk	Phylogenetic tree of the WoL project
tax2tree/	Directory containing the tax2tree results from the WoL project
place/	Output directory of phylogenetic placement.
